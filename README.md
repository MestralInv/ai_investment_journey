
# AI Investment Journey

Welcome! This is the code companion to the LinkedIn newsletter **AI Investment Journey**.

**What you'll find**
- Reproducible notebooks on AI/ML/DL with an investment focus
- Small, clean examples you can run locally
- Reusable utilities (plots, metrics, data helpers)

**How to use**
```bash
git clone https://github.com/<your-handle>/ai-investment-journey
cd ai-investment-journey
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
jupyter lab
